# Shaurya Voicebot

A clean, iMessage-style AI voice assistant built using modern web tech.

## 🚀 Tech Stack
- React + Vite
- Tailwind CSS + DaisyUI
- Web Speech API (voice input/output)
  
## 📦 Getting Started

```bash
npm install
npm run dev
```

## 🔧 Features
- 🎤 Voice-to-text input
- 💬 Typing indicator
- 💻 iMessage-like UI
- 🌐 Works on all devices
